# [RESTful API] 台南旅遊美食地圖

- 說明

  使用 Leaflet + OpenStreetMap 地圖應用做開發，結合台南市政府的開放資料，使想去台南旅遊的遊客能快速找到最適合的資訊。

- 技術

  HTML + CSS + jQuery + Leaflet
